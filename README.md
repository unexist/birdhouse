# Birdhouse

This is fun project to create small birdhouse, attached with motion sensor and cam.

# Parts list

- 1x Raspberry Pi Zero W
- 1x HC SR501 Pir motion sensor
- 1x Pi cam
- 1x Cam cable for Pi Zero
- A bit of wiring..
